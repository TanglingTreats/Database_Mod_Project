1. Data should be created at excel first, due to data in csv format can be tough to change due to configuration
2. From the sql script, you can actually open out ERD diagram. File -> Database -> Reverse Engineer
3. You can also translate from ERD diagram back to script by doing forward Engineer
4. From your excel format you can convert it to csv format by, file -> saved as -> csv UT-8
5. When inserting date into excel, rmb to change saved date format to yyyy/mm/dd. 
Select cell -> format cell -> custom -> enter the date format. 
(Take note, format of cell at least for date, can only be change at excel not csv.
6. You can insert csv data into table, rmb to double check and remove any unwanted column
